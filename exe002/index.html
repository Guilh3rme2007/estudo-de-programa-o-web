Anotações

Novos elementos semânticos

Negrito e itálico

Para colocar termos de forma semântica, utilizamos elementos como <strong> para negrito e <em> para itálico.

Texto marcado

Para deixar o texto marcado utilizamos a tag <mark>.

Texto pequeno

Para criar um texto pequeno usamos a tag <small>.

Texto deletado

Para criar um texto deletado usamos a tag <del>.

Texto inserido

Para criar um texto inserido usamos a tag <ins>.

Texto sobrescrito

Para criar um texto sobrescrito usamos a tag <sup>.
x90+6 - 26 = x90

Texto subscrito

Para criar um texto subscrito usamos a tag <sub>.
CO2

Texto ao contrário

Para criar um texto ao contrário usamos a propriedade CSS direction: rtl;.

Ou posso usar a função <bdo dir="rtl"> para deixar o texto ao contrário.

Outras formatações

Código fonte

O comando <code> deixa o texto com fonte de código e monoespaçada. 

        
    #include 
    int main() {
        printf("Hello, World!");
        return 0;
    }

Pré formatado

O comando <pre> deixa o texto pré-formatado, mantendo espaços e quebras de linha.

        
    #include 
    int main() {
        int cont = 0;
        while (cont <= 100){
            printf("%d \n", cont);
            cont ++;
        }
        return 0;
    }
