Estrutura Básica de referências do documento

As páginas referidas como p1 e p2 apresentam uma estrutura HTML básica, que se inicia com o <!DOCTYPE html>
Essa declaração é fundamental para que o navegador interprete corretamente o documento como HTML5.

Elementos de definição do documento

Em seguida vem o elemento <html lang="pt-br">, que define o idioma da pagina como o português.

Logo após, temos o elemento <head>, que contém as informações meta, título e estilo da página
Nas informações meta temos as seguintes tags:
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Essas tags são importantes para a definição do conjunto de caracteres e para a responsividade da página.

O título da página é definido pelo elemento <title>, sendo a parte que aparece na aba do navegador.

Seguido do icone que aparece no navegador <link rel="shortcut icon" href="imagens/icones/apple-touch-icon.png" type="image/x-icon">

Estilo da página

O estilo da página é definido pelo elemento <style>, que contém as regras de formatação CSS.
Onde se encontram as informações de estilo, como cores, fontes e layout da pagina.
Sendo ligados ao corpo body, aos cabeçalhos h1, h2, h3, aos paragrafos <p> e outros elementos.

Corpo da página

O corpo da página é definido pelo elemento <body>, que contém todo o conteúdo visível para o usuário.

Ambas as páginas utilizam o elemento background (tanto background como background-color).
A primeira página utiliza uma imagem de fundo, enquanto a segunda página utiliza uma cor de fundo sólida.
A função do background é definir a área de fundo de um elemento, podendo ser uma cor sólida ou uma imagem.

Além disso, o corpo da página pode conter diversos elementos, como cabeçalhos, parágrafos, imagens e links, que compõem o conteúdo apresentado ao usuário.

Esses elementos são organizados em uma estrutura hierárquica, permitindo que o navegador interprete e exiba o conteúdo de forma adequada. Sendo divididos em partes relacionadas à apresentação e à estrutura do documento.
Exemplo: um cabeçalho (<h1>) pode conter um título e um subtítulo, enquanto um parágrafo (<p>) pode conter texto e imagens.

Para acrescentar elementos como imagens, utiliza-se a tag <img>, que permite a inserção de imagens no conteúdo da página.

Para acrescentar elementos como emojis, utiliza-se a tag <&#x;> com o código Unicode correspondente ao emoji desejado 📓 .

Para acrescentar elementos como links, utiliza-se a tag <a>, que permite a criação de hyperlinks para outras páginas ou sites.

Para tornar um texto com função semântica, utiliza-se a tag <strong> para negrito e a tag <em> para itálico.

Para criar um texto marcado com função semântica, utilizamos a tag <mark>.

Para criar um texto pequeno semanticamente utilizamos a tag <small>.

Para criar um texto subscrito de forma semântica utilizamos a tag <sub>.

Para criar um texto sobrescrito de forma semântica utilizamos a tag <sup>.

Para colocar um texto ao contrário usamos a propriedade CSS direction: rtl;.
Ou posso usar a função <bdo dir="rtl"> para deixar o texto ao contrário.

Para deixar um texto 

pré-formatado
utilizamos a tag <pre>.

Para deixar um texto

em bloco
utilizamos a tag <blockquote>.

Para realizar uma citação simples, utilizamos a tag <q>.

Para citar Abreviações, utilizamos a tag <abbr>.

Para fazer uma lista de itens, utilizamos a tag <ul> para listas não ordenadas e <ol> para listas ordenadas.

    Lista não ordenada

  • Item 1
  • Item 2
  • Item 3

    Lista ordenada

  1. Item 1
  2. Item 2
  3. Item 3

observação: para adicionar os itens em uma lista, basta utilizar a tag <li> dentro da tag <ul> ou <ol>.

Para criar uma lista de definição, utilizamos a tag <dl>, que permite agrupar termos e suas definições.
